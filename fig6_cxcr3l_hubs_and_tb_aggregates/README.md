# Figure 6 — CXCR3L\(+\) Hubs and Dense TB\(+\) Aggregates

This directory contains the workflow used to compare CXCR3L\(+\) immune hubs to dense T and/or B cell (TB\(+\)) aggregates and to quantify how these structures differ in composition, localization, and association with pathology regions.

Analyses here build on the tessera-based tissue tiling, CXCR3L\(+\) hub annotations, and TB\(+\) aggregate annotations defined in earlier figures to dissect how these structures relate to tumor, stromal, and lymphoid environments.
